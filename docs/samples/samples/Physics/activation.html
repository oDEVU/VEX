Activation ​

It simply defines when object can have its transform recalculated based on physics simulation. If no active forces act on it it gets deactivated (Goes to sleep) to not waste resources. You can also manually specify the activation state of an object. Methods like AddForce, AddImpulse, automatically activate the object if it was deactivated.

Methods ​

cpp
GetPhysicsSystem()->SetBodyActive(bodyID, active);
GetPhysicsSystem()->GetBodyActive(bodyID);
