TransformComponent #

TransformComponent is a component that defines the position, rotation, and scale. It also handles parenting logic and provides methods like getWorldPosition, or GetFrontVector.

Creation #

Its constructor is overloadable, you can pass it some starter values or create empty transform for later use.

cpp
// both definitions are valid
vex::TransformComponent firstTransform = vex::TransformComponent{
    m_engine->getRegistry(),
    glm::vec3{0.0f, 0.3f, 0.0f},
    glm::vec3{0.0f, 0.0f, 0.0f},
    glm::vec3{1.0f, 1.0f, 1.0f}};

vex::TransformComponent seccondTransform = vex::TransformComponent{m_engine->getRegistry()};

Usage #

As said it not only holds data but provides usefull methods.

cpp
myTransform.getForwardVector();

Returns normalized front vector of an object, usefull for example for movement logic. It also implements similar methods for right and up vector.

There are also Get Local and World Position, Rotation and Scale methods, super usefull to get actual in world transforms for child objects.

cpp
myTransform.getWorldPosition();
myTransform.getLocalPosition();

There also are methods to set or add to local or world transforms.

cpp
myTransform.setLocalPosition(glm::vec3{1.0f, 2.0f, 3.0f});
myTransform.addLocalRotation(glm::vec3{1.0f, 2.0f, 3.0f});
myTransform.setWorldScale(glm::vec3{1.0f, 2.0f, 3.0f});
myTransform.addYaw(deltaTime);

WARNING

Manipulating transforms of an physics object may result in zeroing angular and linear velocity.
