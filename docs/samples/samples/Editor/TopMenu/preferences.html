Preferences Menu ​

The Preferences menu provides options of customization for your personal workspace and project identity

Editor Settings ​

  • showFps - checkbox that toggles the display of the FPS counter in the editor

  • showCollisions - checkbox that toggles the display of collisions in the editor

WARNING

This feature is highly resource-intensive and may significantly decrease FPS depending on your hardware


Here you can set different preferences for the editor like:

  • assetBrowserThumbnailSize - changes the size of the thumbnails in the asset browser

  • editorCameraFov - changes the field of view of the editor camera

  • editorCameraRenderDistance - that simply changes the distance at which the camera renders objects

  • frame limit - changes the frame limit of the editor camera

  • vsync - turns on or off vertical synchronization

Project Settings ​

  • project_name - text input that allows you to change the name of the project

  • main_scene - text input that allows you to change the name of the main scene

  • icon_path - text input that allows you to change the path of the app icon (expected format: .png)

  • version - text input that allows you to change the version of the project

  • window_type - dropdown menu that allows you to change the type of the window (windowed, fullscreen, borderless)

  • resolution_mode – dropdown menu that allows you to change the resolution mode

TrybOpis
NativeUses native resolution of the monitor (full resolution without scaling)
RES_240pSimulates 320×240 resolution (typical for old console and arcade games) Renders at low resolution, then scales to full screen with nearest-neighbor filter for pixelated look
RES_480pSimulates 640×480 resolution. Renders at this resolution and scales to screen, maintaining sharp, retro pixel style
PS1_SHARPSpecial mode emulating look of PlayStation 1 games – renders at low resolution (~320×240), applies light filtering and characteristic "wobbling" vertices and affine texture mapping for authentic retro effect
  • vsync - checkbox that toggles vertical synchronization

NOTE

Changes are applied automatically so you don't have to click anything or restart the editor
