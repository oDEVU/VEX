Detailed Description

Interface Class for vulkan backend.

This class manages whole vulkan backend, from initalization, to holding resources, binding window, to calling renderer. Its managing class for backend abstraction. Its architecture allows me to replace it with for example OpenGL Interface with same methods and interface should manage backend by itself.
