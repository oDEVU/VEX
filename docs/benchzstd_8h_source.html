147/* srcBuffer - data source, expected to be valid compressed data if in Decode Only Mode
148 * srcSize - size of data in srcBuffer
149 * fileSizes - srcBuffer is considered cut into 1+ segments, to compress separately.
150 * note : sum(fileSizes) must be == srcSize. (<== ensure it's properly checked)
151 * nbFiles - nb of segments
152 * cLevel - compression level
153 * comprParams - basic compression parameters
154 * dictBuffer - a dictionary if used, null otherwise
155 * dictBufferSize - size of dictBuffer, 0 otherwise
156 * displayLevel - see BMK_benchFiles
157 * displayName - name used by display
158 * @return:
159 * a variant, which expresses either an error, or a valid result.
160 * Use BMK_isSuccessful_benchOutcome() to check if function was successful.
161 * If yes, extract the valid result with BMK_extract_benchResult(),
162 * it will contain :
163 * .cSpeed: compression speed in bytes per second,
164 * .dSpeed: decompression speed in bytes per second,
165 * .cSize : compressed size, in bytes
166 * .cMem : memory budget required for the compression context
167 */
