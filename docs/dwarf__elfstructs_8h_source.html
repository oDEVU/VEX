31/* Typed in from the SystemV Application Binary Interface
32 but using char arrays instead of variables as
33 for reading we don't need the struct members to be
34 variables. This simplifies configure.
35
36 https://www.uclibc.org/docs/elf-64-gen.pdf used as source
37 of Elf64 fields.
38
39 It is expected code including this will have included
40 an official <elf.h> (for various definitions needed)
41 before including this. But that is not strictly necessary
42 given other headers.
43
44 The structs were all officially defined so files
45 could be mapped in. Fields are arranged so
46 there will not be gaps and we need not deal with
47 alignment-gaps.
48*/
49
